# customer-segmentation-app
This project performs customer segmentation using RFM (Recency, Frequency, Monetary) analysis and KMeans clustering. It includes data preprocessing, feature engineering, model evaluation using the Elbow Method and Silhouette Score, and visualization using PCA. The project is deployed as an interactive Streamlit web application.
# 🛍️ Customer Segmentation using RFM & KMeans

## 📌 Project Overview
This project segments customers based on purchasing behavior using RFM analysis and KMeans clustering.

## 🚀 Features
- Data Cleaning
- RFM Feature Engineering
- StandardScaler
- KMeans Clustering
- PCA Visualization
- Streamlit Web App

## 📊 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib
- Streamlit

## 📉 Output
Customers are grouped into clusters such as:
- High Value Customers
- Loyal Customers
- At Risk Customers
